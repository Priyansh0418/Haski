## 🧪 Quick Test Flow

```
1. npm run dev                    (start frontend)
2. Visit http://localhost:5173
3. Click "🚀 Get Started →"
   OR
   Sign up: Create account with email + password
   → Redirects to /dashboard ✅

4. From dashboard:
   • Click "📸 New Analysis"
   • Allow camera permission
   • Take photo → See ResultCard
   • Click "Get Recommendations"
   • See routines, products, diet sections

5. Back to dashboard:
   • See line chart with past 5 analyses
   • Click "🧴 Set Reminder" → Set time + allow notifications
   • Click "⚙️ Settings" → Clear data, toggle privacy

6. Try protected route:
   • Clear authToken from localStorage
   • Go to /dashboard
   • Redirects to /login ✅
```
